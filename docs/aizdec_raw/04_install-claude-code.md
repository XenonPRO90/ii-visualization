VS Code стоит. Теперь добавим в него расширение Claude Code, которое превращает обычный редактор в AI-помощника.

[ ШАГ 1: ОТКРЫТЬ МАГАЗИН РАСШИРЕНИЙ ]

Открой VS Code

Слева найди иконку с четырьмя квадратиками — Extensions (Расширения)

Кликни на неё

Откроется панель поиска расширений.

💡 Горячая клавиша:
Mac: Cmd + Shift + X
Windows: Ctrl + Shift + X

[ ШАГ 2: НАЙТИ И УСТАНОВИТЬ ]

В поле поиска напиши Claude Code

Найди расширение от Anthropic (официальное)

Нажми синюю кнопку Install.

Установка займёт несколько секунд. После этого появится иконка Claude — оранжевый логотип. Она может быть слева в боковой панели или справа вверху.

⚠️ Важно:
Убедись что устанавливаешь именно от Anthropic. Есть подделки с похожими названиями. Смотри на издателя, должно быть написано "Anthropic".

[ ШАГ 3: АВТОРИЗАЦИЯ ]

После установки нужно войти в свой аккаунт Anthropic. Заранее зарегистрируй его и оплати подписку.

Предупреждение: 
Потребуется очень хороший VPN и покупка зарубежной сим-карты. Напиши ассистенту, он подскажет где их можно добыть. Большинство стандарных VPNов не работают с профессиональными инструментами типа Claude Code.
Как открыть панель Claude Code

Способ 1 — через Command Palette (рекомендую):

Нажми Cmd + Shift + P (Mac) или Ctrl + Shift + P (Windows)

Напиши: Claude Code: Open in Side Bat

Нажми Enter — панель откроется справа




Способ 2 — через иконку:

Найди иконку Claude (оранжевый логотип), она может быть в боковой панели слева или справа

Напиши /login.




Авторизация:

В открывшейся панели нажми кнопку Sign In

Откроется браузер — войди в свой аккаунт Anthropic (или создай новый)

Подтверди доступ для VS Code

После этого вернись в VS Code — ты авторизован.

💡 Совет:
Если иконка Claude пропала или ты не можешь её найти, используй Command Palette (Cmd/Ctrl + Shift + P → "Claude Code: Open in sidebar"). 
Либо есть комбинация клавиш, на маке это:
Shift + Command + Escape

[ ШАГ 4: ПРОВЕРКА ]

Убедимся что всё работает:

Открой панель Claude Code (иконка с оранжевым логотипом)

Справа откроется чат, внизу поле для ввода текста

Напиши: Привет

Нажми Enter

Если Claude ответил — поздравляю, установка завершена!




Проверка:

✅ Иконка Claude Code появилась (справа вверху)

✅ Панель чата открывается справа

✅ Ты авторизован 

✅ Claude отвечает на сообщения

Всё так? → Переходи к следующему уроку.

[ ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ ]

Проблема: Claude не отвечает или показывает ошибку

Проверь интернет — открой google.com. Если не грузится — проблема с сетью

Проверь подписку — зайди на claude.ai, посмотри активна ли подписка Pro/Max

Перезапусти VS Code — закрой полностью и открой заново

Переустанови расширение — удали Claude Code и установи заново




Проблема: Кнопка Sign In не появляется

Закрой VS Code полностью

Открой заново

Проверь свой VPN и качество сети




Проблема: браузер открылся, но авторизация не работает

Убедись что входишь в тот же аккаунт Anthropic где есть подписка

Если есть несколько аккаунтов — выйди из всех и войди в нужный

Попробуй другой браузер




💭 От Тома:
Если что-то пошло не так — не паникуй. РЕШАЕТСЯ ВСЁ. Если не получается, открой ChatGPT или Gemini, опиши проблему, скинь скриншотики. Когда голова закипит, перекури, или чайку попей. И опять за дело. Если дело в блокировках, уточни у ассистента что делать.
