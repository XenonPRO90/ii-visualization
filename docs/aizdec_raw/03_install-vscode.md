VS Code это программа, в которой живёт Claude Code. Думай о ней как о рабочем столе для своих проектов. Сама по себе она бесплатная и ставится за 2 минуты.

[ ДЛЯ MAC ]
ШАГ 1: Скачать

Открой браузер и перейди на code.visualstudio.com

Нажми большую синюю кнопку Download for Mac

Скачается файл типа VSCode-darwin-arm64.zip (или похожий)

ШАГ 2: Установить

Открой папку Загрузки (Downloads)

Найди скачанный .zip файл и дважды кликни — он распакуется

Появится файл Visual Studio Code с синей иконкой

Перетащи его в папку Программы (Applications)

💡 Зачем в Программы:
Можно запускать и из Загрузок, но тогда при каждом обновлении будут проблемы. В Программах всё работает как надо.

ШАГ 3: Запустить

Открой Finder → Программы (Applications)

Найди Visual Studio Code и дважды кликни

Может появиться предупреждение "Программа загружена из интернета" — нажми Открыть.




Проверка:

✅ Открылось окно VS Code с тёмным интерфейсом

✅ Слева панель с иконками

✅ В центре приветственная вкладка

Всё так? → Переходи к следующему уроку.

[ ДЛЯ WINDOWS ]
ШАГ 1: Скачать




Открой браузер и перейди на code.visualstudio.com

Нажми большую синюю кнопку Download for Windows

Скачается файл типа VSCodeUserSetup-x64.exe

ШАГ 2: Установить

Открой папку Загрузки

Запусти скачанный .exe файл

Появится установщик — жми Далее на каждом шаге

На шаге "Выберите дополнительные задачи" поставь галочку Добавить в PATH

Нажми Установить и дождись завершения

⚠️ Важно:
Галочка "Добавить в PATH" — обязательна. Без неё Claude Code не сможет нормально работать. Если пропустил, переустанови.

ШАГ 3: Запустить

Нажми Win (кнопка с флажком Windows)

Напиши VS Code или Visual Studio Code

Кликни на появившуюся программу

Проверка:

✅ Открылось окно VS Code с тёмным интерфейсом

✅ Слева панель с иконками

✅ В центре приветственная вкладка

Всё так? → Переходи к следующему уроку.




💭 От Тома:
VS Code это не просто текстовый редактор. Это среда, где ты будешь проводить большую часть времени, работая над проектами. Привыкай к ней. Через пару дней она станет родной.
